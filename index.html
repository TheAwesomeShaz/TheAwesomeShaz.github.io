PROJECTS

CYBERBIRD

Cyberbird is a 3D Third person mistake Hoverboard Simulator which is available as a downloadable game for windows. This game was made in 2 days for the GMTK (GameMaker's Toolkit) 2021 gamejam. In this project. I worked on the Gameplay programming and worked with a team from different time zones including a 3D artist,voice actor and Sound Designer and another programmer. Also worked on managing the whole team. Unity3D was used for game development. I learned how to set up and use unity's ragdolls in this project, and also learned how to manage time effectively when working in a team.

INTIQAAM

Intiqaam is a Third person Stealth game which is available as a downloadable game for windows. This game was made in 3 days for the ludum dare gamejam. This project was greatly inspired by assassins creed's bush/vegetation stealth mechanic. I worked on the programming and sound design, and used an asset pack for the 3D models. Unity3D was used for game development and FLStudio for sound design. I learned how to set up Third person movement and use animation states how to use paired animations.

LOADED DUNGEON

Loaded Dungeon is a sidescroller maze runner which is available as a downloadable game for windows. This game was made in one week for my college mini project. I worked on the programming, sound design as well as some of the pixel art except the environment which is an open source asset. Unity3D was used for game development and FLStudio for sound design. I learned about the new input system and how to take input from mouse and keyboard simultaneously, I also learned how to apply chromatic abberation and other kinds of post processing effects.

CONTROL'NT

Control'nt is a top down on rails shooter which is available as an in-browser game and android downloadable game. The theme for the jam was "out of control". This game was made in 2 days for the GMTK gamejam. In the game, I implemented a change in direction and music every random seconds, the score goes up or down by a random amount, basically everything is out of the player's control except the shooting. The music has 4 versions which is specified for the 4 directions the player will be moving in. Unity3D was used for game development and FLStudio for sound design. I learned how to slow down time in Unity and bring it back to normal.
